# Basic Java Programs

This directory contains fundamental Java programming examples covering basic concepts.

## What Goes Here

- **Variables and Data Types**: Examples showing different data types (int, double, String, boolean, etc.)
- **Control Structures**: If-else statements, switch cases, for loops, while loops
- **Arrays**: Array manipulation, multi-dimensional arrays
- **Strings**: String operations and manipulations
- **Methods**: Function definitions, parameter passing, return types
- **Exception Handling**: Try-catch blocks, custom exceptions
- **File I/O**: Reading from and writing to files
- **Basic Input/Output**: Scanner usage, System.out operations

## Example Project Structure

Each program should be organized like this:
```
program-name/
├── src/
│   └── ProgramName.java
└── README.md
```

## Naming Convention

- Use descriptive names for programs (e.g., `calculator`, `grade-manager`, `file-reader`)
- Follow Java naming conventions (PascalCase for class names, camelCase for variables)

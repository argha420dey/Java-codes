# Algorithms and Data Structures

This directory contains implementations of common algorithms and data structures in Java.

## What Goes Here

### Sorting Algorithms
- Bubble Sort, Selection Sort, Insertion Sort
- Merge Sort, Quick Sort, Heap Sort
- Radix Sort, Counting Sort

### Searching Algorithms
- Linear Search, Binary Search
- Depth-First Search (DFS)
- Breadth-First Search (BFS)

### Data Structures
- **Linear**: Array, LinkedList, Stack, Queue, Deque
- **Trees**: Binary Tree, Binary Search Tree, AVL Tree, Red-Black Tree
- **Graphs**: Adjacency Matrix, Adjacency List representations
- **Hashing**: Hash Tables, Hash Maps
- **Heaps**: Min Heap, Max Heap, Priority Queue

### Advanced Topics
- Dynamic Programming solutions
- Greedy algorithms
- Backtracking algorithms
- Graph algorithms (Dijkstra's, Floyd-Warshall, etc.)

## Project Structure

```
algorithm-or-structure-name/
├── src/
│   ├── Algorithm.java
│   └── TestAlgorithm.java
├── README.md
└── examples/
    └── sample-input.txt
```

## Implementation Guidelines

- Include time and space complexity analysis
- Provide test cases and examples
- Add comments explaining the algorithm logic
- Include both iterative and recursive versions where applicable
